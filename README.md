Corona Awareness Web Application
Overview

Corona Awareness is a responsive web application designed to educate users about COVID-19, its symptoms, preventive measures, and vaccination updates. This project leverages modern web technologies to deliver an interactive and user-friendly interface, making reliable information accessible to everyone.

Features

Informative Dashboard: Displays COVID-19 symptoms, preventive tips, and safety guidelines.

Live Updates: Shows real-time COVID-19 statistics using public APIs.

Responsive Design: Works seamlessly on desktop, tablet, and mobile devices.

User Interaction: Provides search functionality for symptoms, nearby hospitals, and vaccination centers.

Accessibility: Focused on clear visuals, readable content, and smooth navigation for all users.

Tech Stack

Frontend: HTML5, CSS3, JavaScript (ES6)

Frameworks & Libraries: Bootstrap 5 for responsive UI, Chart.js for data visualization

API Integration: Public COVID-19 APIs for live statistics

Version Control: Git & GitHub

Installation

Clone the repository:

git clone https://github.com/<your-username>/corona-awareness.git


Navigate to the project folder:

cd corona-awareness


Open index.html in your browser or use a live server extension in VS Code.

Usage

Launch the application in your web browser.

Navigate through sections: Home, Symptoms, Prevention, Live Updates, and Vaccination.

Search for specific information or explore COVID-19 statistics interactively.
